# Rocket-Music（音乐播放器）
技术栈：vue + vue-router + vuex + Axios

使用网易云音乐NodeJS版API，API地址：https://binaryify.github.io/NeteaseCloudMusicApi/

使用图片懒加载，Webpack分包等优化页面渲染时间，增加用户体验

包含搜索历史，在线搜索，收藏歌曲，随机播放等多功能交互
